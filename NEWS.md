# richStudio News

## Changes in version 0.1.5 (2025-12-05)

### New Features
* Initial Bioconductor submission
* Multiple clustering methods: richR (Kappa-based), Hierarchical, and DAVID-style
* Dynamic parameter UI that adapts to selected clustering method
* Session save/load functionality (RDS and JSON formats)
* File management: rename and remove uploaded datasets
* Interactive visualizations: bar plots, dot plots, heatmaps, and network graphs
* Support for GO, KEGG, and Reactome enrichment analysis via richR and bioAnno

### Infrastructure
* Shiny module architecture for maintainable code
* C++ backend for efficient Kappa similarity calculations
* Comprehensive dependency management with fallback installation options

### Documentation
* Full roxygen2 documentation for exported functions
* Vignette with usage examples
* README with installation and quick start guide
